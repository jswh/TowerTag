# TowerTag
[Tower](https://tower.im/)的标签系统增强插件。包括一些预定义功能标签和自定义标签高亮

## 安装

从chrome的[应用商店](https://chrome.google.com/webstore/detail/tower-tag/pkdloleicnhgciodnffcfeimpoialjai?hl=en-US)安装

##功能

###功能标签

* block-header  分割一个list的特殊标记。

###自定义标签高亮
